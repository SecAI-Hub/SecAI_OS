# Support

## Usage Questions

For general questions about using SecAI OS:

- **GitHub Discussions** -- [SecAI-Hub/SecAI_OS Discussions](https://github.com/SecAI-Hub/SecAI_OS/discussions)
- Check the [FAQ](docs/faq.md) for common questions
- Review the [documentation](docs/) for guides and references

## Bug Reports

Found a bug? Please open a [GitHub Issue](https://github.com/SecAI-Hub/SecAI_OS/issues/new) with:

- SecAI OS version (commit hash or image tag)
- Hardware details (GPU vendor/model, RAM, storage)
- Steps to reproduce
- Expected vs actual behavior
- Relevant logs (`journalctl -u secure-ai-<service>`)

## Security Issues

**Do not report security vulnerabilities via GitHub Issues.**

See [SECURITY.md](SECURITY.md) for responsible disclosure instructions.

## Feature Requests

Have an idea for improvement? Open a [GitHub Issue](https://github.com/SecAI-Hub/SecAI_OS/issues/new) with the "enhancement" label. Include:

- What problem it solves
- Proposed solution
- Alternatives you considered
